Testing Easel

Hello! This is a mock website that mimics the structure of the University of Auckland lecture recording interface.

It has been created to test Easel without using University infrastructure or needing a University issued account.

This website recreates the following pages:

  • LTR (Lecture Threatre Recordings) - An overview of all lecture recordings
  • Individual Recording Pages - A page containing a video element

To use, please click the green "Open with Easel" button in the iframe below.

Alternatively, this page can be viewed directly.
